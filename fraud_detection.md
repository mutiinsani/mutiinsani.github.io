## Fraud Detection on Credit Card Transaction
---
### Project description

Developing a model to predict whether a transaction is fraudulent or not.
We will train a model with past transactions data. There is not data of the person doing the transaction only transaction related data.

### Dataset
- Historical transactions consist of real and fraudulent data. This dataset is highly imbalance.

### Process
General process consist of:
1. Data preparation
    - Data cleaning
    - Feature extractions
2. Modeling
    - SVM
    - Logistic Regression
    - Gradient Boosting
3. Evaluation
